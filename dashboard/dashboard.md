# 🤖 Bot Management Dashboard

## A web-based dashboard that allows you to easily manage your bots in one place. This system provides powerful tools for monitoring, controlling, and interacting with your bots through a simple and user-friendly interface.
### 🚀 Features

# 📊 Dashboard Overview
- View real-time statistics of your bots
- Monitor activity and performance

# 👥 User Management
- Ban / Unban users
- Manage user permissions and roles

# 📩 Messaging System
- Send messages directly via the dashboard
- Communicate with users or bots instantly

# 📈 Statistics & Logs
- Track usage data
- View logs and bot activity

# ⚙️ Bot Controls
- Manage bot settings
- Enable / disable features
